The workshops

Before staring a workshop, check its requirements. Some of these will be practical, such as setting up a development environment, others require an understanding of web fundamentals like HTML, browser development tools, and HTTP.

You can take workshops in any order but each workshop makes assumptions about your knowledge. If you are well versed in HTML and CSS, for example, you can skip the first workshop.

But if your knowledge or understanding is partial, we recommend taking Workshop one before Workshop two.
